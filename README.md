#  HR Analytics Dashboard Using Power BI

##  Project Overview

This repository contains an **interactive HR Analytics Dashboard** built using **Microsoft Power BI**. The dashboard helps visualize and analyze human resources data to support decision-making in areas such as employee attrition, demographics, and workforce trends. It’s designed for HR managers, business analysts, and data enthusiasts who want data-driven insights into workforce patterns.

---

##  Key Features

✔ Interactive Power BI visuals such as charts, tables, and filters  
✔ Attrition analysis by age, department, gender, years at company, etc.  
✔ Employee demographics overview  
✔ Trend analysis and key HR metrics visualization  
✔ Exportable reports and visuals for presentations

##  Dashboard Highlights

The dashboard contains the following insights:

- **Employee Count & Attrition Rate**
- **Gender-wise Distribution**
- **Age Group Analytics**
- **Department & Job Role Trends**
- **Slicers & Filters for Interactive Exploration**

The visuals enable you to explore employee data and investigate patterns related to retention, demographics, and workforce composition.
